Sistema de Gestão de Estoque
Sistema desktop para gerenciamento de estoque de uma loja de peças, desenvolvido individualmente com interface gráfica e integração com banco de dados.

Tecnologias:

Python — lógica do sistema e interface gráfica
Tkinter — interface gráfica desktop
MySQL — banco de dados relacional


Funcionalidades:

Cadastro e consulta de produtos
Controle de entrada e saída de estoque
Visualização do estoque atual
Interface gráfica intuitiva com menu principal


Pré-requisitos:
Python 3.x instalado
MySQL instalado e rodando
Biblioteca mysql-connector-python
